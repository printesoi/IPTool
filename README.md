To be able to compile this app you need Qt headers, qmake and g++

For archlinux:
        sudo pacman -S qt g++

For [.]Ubuntu:
        sudo apt-get install qt4-qmake libqt4-dev g++

Just run:
        qmake && make

##Usage:##
    Control-Q -> Quit
    Control-L -> Clear the IP
    F1        -> Display "About"

    You can click on the bits below the input fields to change the values.
